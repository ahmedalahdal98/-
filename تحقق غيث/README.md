# بوابة التحقق

واجهة عربية جاهزة للتجربة تعمل مباشرة في المتصفح بدون تثبيت.

## التشغيل

افتح `index.html` مباشرة، أو شغّل خادمًا محليًا داخل المجلد:

```powershell
python -m http.server 8000
```

ثم افتح `http://localhost:8000`.

## حسابات التجربة

- الموظفة سارة أحمد: `1111`
- الموظف محمد خالد: `2222`
- المدير: `admin123`

النسخة الحالية تحفظ البيانات في `localStorage` للتجربة. عند النشر الفعلي يجب ربطها بقاعدة بيانات وخادم خلفي، وتخزين كلمات المرور بصورة مشفّرة.
